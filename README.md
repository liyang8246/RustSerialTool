# 串口监视器

## 项目简介

本项目是一个基于 Rust、Svelte 和 Tauri 开发的跨平台串口监视器。它可以帮助用户轻松监控和管理串口数据，支持多种波特率、数据位、停止位和校验方式设置，界面简洁，操作便捷。

## 特性

- 跨平台支持（Windows、macOS、Linux）
- 界面美观，操作简单
- 支持多种串口参数设置
- 实时数据显示与保存
- 支持发送数据到串口
- 支持串口数据十六进制显示

### 使用说明

1. 打开应用，选择需要监视的串口。
2. 设置波特率、数据位、停止位和校验方式。
3. 打开串口，开始监视数据。
4. 可以实时查看串口数据，并支持十六进制显示。
5. 支持发送数据到串口。
